///////////////////////////

Making claims about sequences.

Suppose "seq" is our sequence



∀ (lower until upper)(k => some statement about seq(k) )
    (does NOT include upper)





∀ (0 until seq.size)(k => seq(k) == 0 )
    
    --> what does this say? all elements in the sequence are 0





∀ (0 until seq.size)(k => k > 2 → seq(k) == 0 )
    
    --> what does this say? all positions after 2 will be changed to 0






Similarly, can use ∃

∃ (0 until seq.size)(k => seq(k) == 0 )

    --> what does this say? there is a position that has been changed to 0
