Halting problem - given a computer program and a set of inputs
    is it possible to write another program to determine if the original
    program will halt (terminate) for the inputs?


We will use PBC to prove that we CAN'T write such a program.

Assume, for the sake of contradiction, that we CAN write a program
to determine if another program will terminate on a set of inputs.
 That is, assume we have:

 def Halt(Function, Input):
    if (Function(Input) halts) return true
    else return false


We also write:

    def DoOpposite(Function):
        if (Halt(Function, Function) return true): loop forever
        else: return (i.e., halt) 


Consider DoOpposite(DoOpposite). Two cases: 1) halts. 2) loops forever.

Case 1. Suppose DoOpposite(DoOpposite) halts. That means 
Halt(DoOpposite, DoOpposite) returned false. That meant DoOpposite(DoOpposite)
did not halt. Contradiction.

Case 2. Suppose DoOpposite(DoOpposite) loops forever. then
Halt(DoOpposite, DoOpposite) returned true. Then DoOpposite(DoOpposite) halts.
This is another contradiction.


This means it is NOT possible to write a Halt program to determine if another
program will terminate.
